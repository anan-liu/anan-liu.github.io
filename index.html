composition(复合)

在类中可以声明其他的类对象，该类可以使用类对象的属性。关系是”has a”

delegation（委托）

在A类中声明一个B类指针，是一种特殊的复合。A类的所有对象中的B类指针都指向同一个内存空间，当A类对象中某一个需要改变指针所指时，将copy一份出来给对象改动。写时复制技术。

inheritance（继承）

子类继承父类，分为公有继承，私有继承，保护继承。关系是”is a“

虚函数&纯虚函数

在类中使用virtual关键字标识一个成员函数，这个函数就是虚函数，子类可以override虚函数，也即重新定义函数功能。父类中，虚函数可以有默认功能，子类不必override。但是纯虚函数要求子类一定要override，声明纯虚函数的方式就是在虚函数后加上” = 0“

inheritance with virtual function

Template Method 模式是一种行为设计模式，它定义了一个算法的骨架，并将一些步骤的具体实现延迟到子类中。这样可以在不改变算法结构的情况下，允许子类重写部分步骤以提供不同的实现0。

规范写法

防卫式声明

尽可能使用const

如果一个成员函数里头，没有更改过任何数据，理应加上const

double real() const {return re;}

array new使用array delete

声明写在.h文件，定义写在.cpp文件

如果将定义写在.h文件中，那么当这个.h文件被多次包含时将导致重复定义。

.cpp文件不能直接被include，会导致重复定义。

因此应当将声明写在 .h文件中，定义写在.cpp中。main.cpp包含.h文件
