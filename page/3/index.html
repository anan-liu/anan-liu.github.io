explicit

发表于 2023-10-05 更新于 2024-01-08 分类于 C++11

explicit

explicit 是 C++ 中的一个关键字，主要用于修饰类构造函数。使用 explicit 关键字可以防止隐式类型转换，强制要求显式地调用构造函数。

默认情况下，如果一个构造函数可以用作隐式转换，那么它可能会导致一些不直观的行为。explicit 关键字的使用可以帮助避免这种情况。
