防卫式声明

尽可能使用const

如果一个成员函数里头，没有更改过任何数据，理应加上const

double real() const {return re;}

array new使用array delete

声明写在.h文件，定义写在.cpp文件

如果将定义写在.h文件中，那么当这个.h文件被多次包含时将导致重复定义。

.cpp文件不能直接被include，会导致重复定义。

因此应当将声明写在 .h文件中，定义写在.cpp中。main.cpp包含.h文件

返回值

  1. 引用返回
  2. 值返回

返回值尽量使用返回引用值。当函数返回时，任何在该函数声明的变量将被释放掉。如果返回的变量是在该函数声明定义的，或者是该函数中的临时对象，那么不能使用引用返回

友元函数

在类中使用关键字friend修饰某个函数原型，这样，这个函数就能访问私有属性。

同类的objects互为友元，即objects的成员函数也能访问到私有属性。
