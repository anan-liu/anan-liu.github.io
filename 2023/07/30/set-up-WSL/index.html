WSL的安装

介绍

WSL（Windows Subsystem for Linux）是一种由微软开发的技术，它允许在 Windows 操作系统上运行一个本地的 Linux 环境。简单来说，WSL 是一个用于在 Windows 上运行 Linux 发行版的兼容层。

使用WSL，相比于虚拟级，使用WSL更轻量级且集成度更高。

配置

  1. 重启计算机后会弹出窗口，静待它安装。安装可能会出错，图示：

  2. 尝试在powershell使用下列命令：

    Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

    然后重启再次尝试

    详情参阅：WSL Ubuntu error 0x80004002 · Issue #2851 · microsoft/WSL (github.com)
