引用的使用注意事项

  1. 引用一个变量相当于给这个变量起别名，定义引用时不开辟新空间

  2. 声明引用时必须初始化，因为引用不是对象，它只能绑定对象而存在

  3. 引用不是对象，因此指针不可能指向引用，引用的引用也不存在。但是指针是一个对象，所以引用可以引用指针。

复杂类型的确定

确定一个声明语句所声明的变量是什么类型时，使用从右向左阅读的方法，离变量越近的符号对变量的作用越大。如上表达式的r，&离的近，因此它先是一个引用，而后是*，因此它是一个引用指针的引用。

const的引用

当一个普通的整数型变量是const修饰时，为了定义它的引用，引用也必须const修饰。：

const int i = 2048; const int &r = i;

const修饰的引用不再可以修改它所绑定的对象

一般情况下，引用的类型必须和它所引用的变量的类型完全一致。但有例外。
