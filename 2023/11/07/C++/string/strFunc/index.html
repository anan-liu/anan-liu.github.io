发表于 2023-11-07 更新于 2023-11-09 分类于 C++

字符串函数

strlen()

传入参数是一个字符串，返回值是该字符串的长度

strcat()

传入参数是两个字符串，把后一个参数的字符串拼接到前一个参数的字符串之后。

strncat()

传入参数是两个字符串和一个整数值，和strcat()函数行为一样，不过多了一个大小限制，当拼接的字符个数大于这个限制时或遇到空字符时停止。

strcmp()

传入参数是两个字符串，用以比较两个字符串的内容。如果字符串相同则返回0。如果不同，返回第一个不同之处值，该值的计算方法是第一个字符串的字符减去第二个字符串的字符的ASCII值。

strncmp()

和strcmp()行为一样，但是多了一个第三参数，是一个整数值，说明仅需要比较多少个字符即可。

strcpy()

strncpy()

fgets()

gets()由于没有边界检查，在C11中不再被启用，转而使用fgets()。

  • 第一参数是一个字符串，接下来输入或读取的数据存储在这个字符串中
  • 第二参数是一个整数值，限制输入字符串的长度
  • 第三参数是数据来源，使用stdin将会被要求输入数据

fgets()的返回值是一个指向char的指针，地址和第一参数相同。但是如果读到了文件末尾或者是一个空行，将返回空指针。
